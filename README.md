## FizzBuzz!


---
### Usage:
* From bash
```shell
poetry run fizz_buz --help
```
```shell
usage: fizz_buzz [-h]

Console utility to convert number to FizzBuzz notation

options:
  -h, --help  show this help message and exit
```
* From Python:
```python
from fizz_buzz.core import convert_number
a = convert_number(3)  # "Fizz!"
```
